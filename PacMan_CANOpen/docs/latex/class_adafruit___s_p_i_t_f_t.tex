\mbox{\hyperlink{class_adafruit___s_p_i_t_f_t}{Adafruit\+\_\+\+S\+P\+I\+T\+FT}} is an intermediary class between \mbox{\hyperlink{class_adafruit___g_f_x}{Adafruit\+\_\+\+G\+FX}} and various hardware-\/specific subclasses for different displays. It handles certain operations that are common to a range of displays (address window, area fills, etc.). Originally these were all color T\+FT displays interfaced via S\+PI, but it\textquotesingle{}s since expanded to include color O\+L\+E\+Ds and parallel-\/interfaced T\+F\+Ts. Many of the class member functions similarly live on with names that don\textquotesingle{}t necessarily accurately describe what they\textquotesingle{}re doing, again to avoid breaking a lot of other code.
