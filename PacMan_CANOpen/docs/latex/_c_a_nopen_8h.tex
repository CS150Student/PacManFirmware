\doxysubsection{Detailed Description}
Main C\+A\+Nopen stack file.

It combines Object dictionary (C\+O\+\_\+\+OD) and all other C\+A\+Nopen source files. Configuration information are read from C\+O\+\_\+\+O\+D.\+h file. This file uses one C\+AN module. If multiple C\+AN modules are to be used, then this file may be customized for different C\+A\+Nopen configuration. (One or multiple C\+A\+Nopen device on one or multiple C\+AN modules.)
