volatile bool_t CO_CANmodule_t::bufferInhibitFlag

◆ useCANrxFilters

volatile bool_t CO_CANmodule_t::useCANrxFilters

Value different than zero indicates, that CAN module hardware filters are used for CAN reception. If there is not enough hardware filters, they won't be used. In this case will be all received CAN messages processed by software.
